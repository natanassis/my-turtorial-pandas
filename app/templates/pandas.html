Introdução ao Pandas

Vamos iniciar nosso tutorial de Pandas

Para começar, vamos primeiro instalar o Pandas através do pip.

$ pip install pandas

Caso já tenha instalado o Pandas, dentro de um arquivo Python, digite:

  • Primeiro, vamos importar a biblioteca do Pandas

    • # Importando a biblioteca
    >>> import pandas as pd

  • Vamos criar um dicionario em python

    • # criando o dicionario em python
    >>> dados = {
    "nome":["Pedro","João",'Maria',Maila],
    "idade":[23,25,36,25],
    "profissao": ["Aluno","Professor","Atriz",Estudante],
    "Cidade": ["Salvador","Recife","Rio de Janeiro","Salvador"],
    "sexo": ["M","M","S","S"]
    }

  • Feito isso vamos transforma em dataframe no pandas

    • # Transformando o dicionario dados em python em um dataframe pandas

    >>> df = pd.Dataframe(dados)

    IMPORTANTE: Caso queria criar um dataframe,
    por meio de uma unica linha no dicionario será preciso
    usar o index=0 dentro pd.Dataframe(dados,index=0)

  • para tranforma em arquivos Excel usamos o to_excel

    • # Transformando o dataframe pandas em um arquivo excel

    >>> novo_arquivo = df.to_excel("novo_arquivo.xlsx")

    OBS: Existes varios formas de criar arquivos com pandas
    de diferentes tipos também csv, json, etc...

  • Para pegar apenas alguma coluna especifica

    • # Pegando um coluna especifica

    >>> nome_idade = df.[['nome','idade']]
    >>> print(nome_idade)

    Esse método é para pegarmos apenas colunas, mas se você
    quiser pegar apenas uma linhas, ou linhas ,até mesmo um valor especifico

  • Para isso vamos usar o metodo .loc[]

    • # pegar uma linha 2 linhas

    >>> print(df.loc[1:2])

    # para pegar linhas que corresponde a uma codição

    >>> alunos = df.loc["profissao"] = 'Aluno'

    >>> print(alunos)

    # Pegar várias linhas e colunas usando o loc

    >>> cidade_salvador = df.loc[df["cidade"] == "Salvador", ['nome','profissao']

    # Pegar um valor especifico

    >>> print(df.loc[1,'nome'])

  • Vamos adicionar uma coluna.loc[]
    para adcionar uma coluna usaremos como abaixo

    • # Criando uma nova coluna com uma valor padrão

    >>> novo_coluna = df[:,'Descricao'] = "Pessoa Fisica" print(nova_coluna)
    # Criar uma coluna a parti de uma coluna existente

    >>> df.loc['media_idade'] = len(df['idade']) / 5

  • Vamos adicionar uma linhas
    Para adicionar uma linha usaremos o .append

    • # Adicionar uma linhas
    # criando o dicionario em python
    >>> nova_linha = {
    "nome":["Gilberto"],
    "idade":[46],
    "profissao": ["Jardineiro"],
    "Cidade": ["Salvador"],
    "sexo": ["M"]
    }
    >>> novo_linha = pd.Dataframe(nova_linha)
    >>>df = df.append(nova_linha)
    >>>print(df)

  • Excluir linhas e Colunas
    Para excluir linas ou colunas usaermos o .drop

    •                     
                        # Excluindo uma linha
    
                        
    
                        # axis = 0 -> è o eixo da linha
                        
    
                        >>> novo_df = df.drop(0,axis=0)
                        
    
                        print(nova_df)
                        
    
                        # Excluindo uma coluna
    
                        
    
                        # axis = 1 -> è o eixo da coluna
                        
    
                        >>> novo_df = df.drop(1,axis=1)
                        print(nova_df)
